# Movie Booking System (電影訂票系統)

## 專案簡介

這是一個使用 **Java + JavaFX** 開發的電影訂票系統，提供從電影瀏覽、場次與座位選擇，到付款、退票、訂單管理與推薦等一條龍流程。
系統採用分層設計，將資料模型、業務邏輯與 UI 分離，方便維護與功能擴充。

---

## modules 資料夾 (UI 介面模組)

### BookingModule.java
- **用途**：訂票流程介面模組
- **主要功能**：
  - 顯示電影場次選擇介面
  - 提供座位選擇的互動式座位圖
  - 選擇票種（成人、學生、早鳥）與數量
  - 選擇餐點組合（爆米花可樂、情侶套餐等）
  - 即時計算與顯示總金額
  - 引導使用者進入付款流程

### LoginModule.java
- **用途**：使用者登入與註冊介面模組
- **主要功能**：
  - 提供帳號密碼登入介面
  - 密碼輸入遮罩保護
  - 登入驗證與錯誤提示
  - 忘記密碼功能入口
  - 串接 AuthService 進行身份驗證

### OrderModule.java
- **用途**：訂單管理與歷史紀錄介面模組
- **主要功能**：
  - 顯示使用者所有訂單列表
  - 訂單詳細資訊查看（場次、座位、金額等）
  - 訂單狀態顯示（已付款、已取消、已退款）
  - 提供退票功能入口
  - 電子票券查看與顯示

### PriceModule.java
- **用途**：票價查詢與比較介面模組
- **主要功能**：
  - 顯示不同影城的票價資訊
  - 跨影城價格比較功能
  - 依電影、影城、時段篩選票價
  - 價格排序與最優惠方案推薦
  - 折扣碼資訊顯示

### RefundModule.java
- **用途**：退票與退款處理介面模組
- **主要功能**：
  - 選擇要退票的訂單
  - 顯示退款金額與手續費計算
  - 退票條件檢查提示
  - 確認退票操作
  - 退款結果顯示與通知

## services 資料夾 (業務邏輯服務)

### AdminModule.java
- **用途**：系統管理員功能服務
- **主要功能**：
  - 管理電影資料（新增、修改、刪除）
  - 管理影城與場次資訊
  - 查看系統統計資料
  - 折扣碼管理
  - 使用者帳號管理

### BookingService.java
- **用途**：訂票流程核心服務
- **主要功能**：
  - `autoSelectSeats`：根據需求票數自動挑選最佳座位
  - `listAvailableSeats`：列出場次可用座位清單
  - `createBooking`：建立訂單並綁定使用者、場次、座位
  - 計算總價格（票種差異、餐點加價、折扣碼）
  - 座位狀態管理與鎖定機制
  - 訂單查詢與詳細資訊提供

### PriceService.java
- **用途**：票價計算與折扣處理服務
- **主要功能**：
  - 根據票種計算基礎票價
  - 處理折扣碼驗證與折扣計算
  - 餐點組合加價計算
  - 跨影城票價比較邏輯
  - 特殊優惠與促銷價格處理

### UserService.java
- **用途**：使用者資料與帳號管理服務
- **主要功能**：
  - 使用者註冊與資料建立
  - 登入驗證（帳號密碼檢查）
  - 密碼加密與雜湊處理
  - 密碼修改與重設功能
  - 使用者偏好設定管理
  - 使用者訂單歷史查詢

## 如何執行

### 環境需求
- Java JDK 11 或以上
- Maven 3.6 或以上
- JavaFX SDK

### 執行步驟
```bash
# 1. Clone 專案
git clone https://github.com/ximena-maker/Movie-Booking-System.git

# 2. 進入專案目錄
cd Movie-Booking-System/Movie-Booking-System

# 3. 使用 Maven 編譯
mvn clean compile

# 4. 執行應用程式
mvn javafx:run
```
